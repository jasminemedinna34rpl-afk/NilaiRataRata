main fungsi membaca jumlah siswa, membaca tiap nilai, menghitung dan menampilkan rata-rata.
jumlah (int) → jumlah siswa.
total (double) → penampung akumulasi nilai.
nilai (double) → nilai tiap siswa.
rata (double) → hasil rata-rata total / jumlah
Baca jumlah. Jika jumlah <= 0, hentikan atau minta ulang (validasi).
baca nilai dan lakukan total += nilai.
rata = total / jumlah.
Cetak rata.
